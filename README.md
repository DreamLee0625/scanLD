# scanLD
A Novel Method for Scanpath Comparison based on Levenshtein Distance

## Document of script
### scanLD.py
The script of scanLD method.
### demo.py
Use scanLD to run a demo of real scanpath data.
### utils.py
Some of the tools used in the demo.

##
